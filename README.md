# MIETI
Este repositório contém documentação(testes, projetos, livros) das várias unidades curriculares que constituem o curso de Engenharia de Telecomunicações e Informática. Resulta da agregação de vários repositórios que existiram no curso, além disso contém documentação presente em repositórios de outros cursos.
